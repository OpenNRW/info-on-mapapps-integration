Here you will find documented examples.

  • open new tab
    • Load a wms url via mapapps in a new tab
    • open.nrw newTab demo Demo shows an integrated example, which also outlines a jsonld parsing. Adaptions are made from line 303 ff.
  • iframe
    • Load a wms url via mapapps in an iframe
    • open.nrw iframe Demo Demo shows an integrated example of mapapps in open.nrw demo page using the iframe functionality. Additionally jsonld parsing is outlined. Adaptions are made from line 304 ff.
  • div integrationAPI
    • Launch map.apps via JavaScript to a div element and load service url via integrationAPI. Please note that a local running mapapps instance is used in the implementation as we need to ensure that needed bundles are available. Of course, the overall mechanism can be adopted to another mapapps host.
    • open.nrw integrationAPI Demo Demo shows an integrated example of mapapps in open.nrw demo page using the integrationAPI. Service URL is determined via jsonld parsing approach. Please note that a local running mapapps instance is used in the implementation as we need to ensure that needed bundles are available. Adaptions are made from line 304 ff.
    • open.nrw integrationAPI Demo with geoportal app Demo shows an integrated example of mapapps in open.nrw demo page using the integrationAPI by using the geoportal.nrw mapapps instance. Please note that the demo does not work since requirements are not fulfilled (custom bundles are missing). Adaptions are made from line 304 ff.
