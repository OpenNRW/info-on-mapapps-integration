Demo - iframe Integration


Define iframe in DOM and pass iframe Source via html link element

		<!-- Define iframe as dom node element -->
		<iframe id="iframe_loadUrl" width="800px" height="800px" name="iframe_loadUrl">
		<!-- Set href attribute to iframe dom element -->
		<a href="https://www.mapapps-a.nrw.de/standalone-3.9.2/geoportal/?lang=de&serviceURL=http://www.geoserver.hochsauerlandkreis.de/arcgis/services/wasser/WSG/MapServer/WmsServer?REQUEST=GetCapabilities&SERVICE=WMS" target="iframe_loadUrl">Load Geoviewer to iframe</a>

Load Geoviewer source to iframe via link

Create iframe, define its attributes and place it at defined dom-node using javascript function; trigger function using link

		<script type="text/javascript">
		//define function which created new iframe
		var createIFrame = function () {
			var iframe = document.createElement('iframe');
			iframe.src = "https://www.mapapps-a.nrw.de/standalone-3.9.2/geoportal/?lang=de?lang=de&serviceURL=http://www.geoserver.hochsauerlandkreis.de/arcgis/services/wasser/WSG/MapServer/WmsServer?REQUEST=GetCapabilities&SERVICE=WMS";
			iframe.height = "800px";
			iframe.width = "800px";
			//place iframe to target dom node
			document.getElementById("anchorForIFrameUsingCreateIFrameFunction").appendChild(iframe);
		};
		</script>

		<!-- anchor element, where iframe will be placed -->
		<div id="anchorForIFrameUsingCreateIFrameFunction"></div>
		<!-- call defined function -->
		<a> id="createIFrameLink" href="#" onclick="createIFrame();return false;">Create iframe using function</a>
